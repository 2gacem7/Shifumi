# Shifumi

A Shifumi (Rock-Paper-Scissors) game built with ReactJs.

## Getting Started

Play the Shifumi game on various devices, as it is now fully responsive and adaptable to different screen sizes.

### Installing

Enter in the ./project folder.

Installation

    npm install

Start the project

    npm run start
